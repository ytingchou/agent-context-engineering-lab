```text
        ↓

        ↓

```

## Why This Order
1. Prompt Engineering: learn to ask clearly and constrain output format.
2. Context Engineering: learn to control input scope, token budget, and phase-based context flow.
3. Agent Skills: package repeatable prompting behavior.
4. Agent Pipelines: chain skills into stage contracts with quality gates.
5. AI-assisted Software Migration: apply the pipeline on real legacy-to-web transitions.
