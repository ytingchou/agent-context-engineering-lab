# Workflow: Run Artifact Chain Session

1. Open `pipelines/DELPHI-TO-NEXTJS-PIPELINE.md` and confirm the six-stage contract.
2. Open `SESSION-02-DELPHI-TO-NEXTJS-CHAIN.md`.
3. Execute Stage 1 -> Stage 6 prompts sequentially in Cline.
4. Apply each stage's repair prompt if checkpoint fails.
5. Capture final output using `examples/pipeline-output-template.md`.
6. Run `npm run check`.
