# Workflow: Run Chain To Capstone Bridge

1. Complete `SESSION-02-DELPHI-TO-NEXTJS-CHAIN.md` first.
2. Open `SESSION-03-CHAIN-TO-CAPSTONE.md`.
3. Run Phase 1 -> Phase 4 prompts in order.
4. Use repair prompt when checkpoint fails.
5. Score final output with `rubrics/capstone-rubric.md`.
6. Save final output using `examples/capstone-output-skeleton.md`.
7. Run `npm run check`.
