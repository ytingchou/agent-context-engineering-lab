# Workflow: Run Agent Context Engineering Lab

1. Open `README.md` and pick a session pack from `Session Packs`.
2. Execute prompts from Step 01 to Step 10 in order.
3. Validate each step using checkpoint and repair prompt when needed.
4. Fill `sessions/session-record-template.md` during execution.
5. Score capstone output with `rubrics/capstone-rubric.md`.
6. Run `npm run check` to verify repository structure.
