1. Read `LEARNING-JOURNEY.md` to align on capability ladder and migration target chain.
2. Review `pipelines/DELPHI-TO-NEXTJS-PIPELINE.md` stage contracts.
3. Run sessions in recommended order:
session-01 -> session-02 -> session-03 -> session-04 -> session-05 -> session-06 -> session-07 -> session-08 -> session-09 -> session-10 -> session-11 -> session-12.
4. Open `README.md` and execute the selected session pack prompts in order.
5. Validate each step/phase using checkpoint and repair prompt when needed.
6. Fill `sessions/session-record-template.md` during execution.
7. Score capstone output with `rubrics/capstone-rubric.md`.
8. Optionally capture final artifact chain using `examples/pipeline-output-template.md`.
9. Run `npm run check` to verify repository structure.
